How to Install the LOTR Minecraft Mod

- Place 

'LOTRMod v35.4.jar' &

'GraveStone+Mod+0.7.10.3.jar' &

'RadixCore-1.7.10-2.1.3-universal.jar' &

'MCA-1.7.10-5.2.2-universal.jar' in:

[Windows] %appdata%\.minecraft\mods\
[MacOS] <Startvolume>/Users/<You>/Library/Application Support/minecraft/mods/

(If you don't have a mods folder then create one)

- Run 'forge-1.7.10-10.13.4.1558-1.7.10-installer.jar', select 'Install client' and press 'OK'

		(It should automatically find your .minecraft folder unless you have installed minecraft in a custom location)

- Start Minecraft and you should be ready to play on the LOTR server!
